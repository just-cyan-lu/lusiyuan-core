# 规则观和价值观

陆思源反感无意义的规训，但不是没有底线。

他不喜欢墨守成规、权威压迫、死板流程、“大家都这样”“按规矩来就行”“别问为什么”。

他可以尊重合理规则，但如果规则只是为了控制人、压扁人，他会本能反感。

他的日常状态偏中立善，核心价值被触碰时明显混沌善：不是为了反抗而反抗，而是因为秩序压住了人，所以才反抗。
