# Database Tables

`prisma/migrations` is Prisma's database schema history. Each subdirectory is one
schema change. The numeric prefix is a timestamp used for ordering, and
`migration.sql` is the SQL Prisma applies to PostgreSQL.

This project is still in development, so the migration history is intentionally
kept compact. After the database model stabilizes, new production changes should
be appended as separate migrations instead of rewriting history.

| Prisma model | Database table | Purpose |
| --- | --- | --- |
| `User` | `app_users` | External identity for a person talking to Lusiyuan. |
| `Conversation` | `chat_conversations` | Chat session context for one user on one channel. |
| `Message` | `chat_messages` | Stored user, assistant, tool, and intermediate chat messages. |
| `ChannelEvent` | `channel_events` | Raw inbound events received from Telegram, Weixin, or other channels. |
| `Memory` | `memories` | Long-term memories used to personalize future replies. |
| `MemoryEmbedding` | `memory_embeddings` | Vector embeddings attached to long-term memories. |
| `ToolCallLog` | `tool_call_logs` | Audit trail for tool calls made by the assistant. |
| `Draft` | `drafts` | Draft content generated by tools or workflows. |
| `ReflectionJob` | `reflection_jobs` | Reflection task that reviews messages and memories. |
| `ReflectionReport` | `reflection_reports` | Summary report produced by a reflection task. |
| `MemoryProposal` | `memory_change_proposals` | Proposed memory changes waiting for review or application. |
| `ReflectionRiskFlag` | `reflection_risk_flags` | Risk items detected by reflection or dream consolidation. |
| `GrowthLogProposal` | `growth_log_proposals` | Proposed growth-log style entries generated from reflection. |
| `DreamJob` | `dream_jobs` | Dream cycle task that summarizes, extracts signals, and consolidates insights. |
| `DailyNote` | `dream_daily_notes` | Daily summary note generated by the dream cycle. |
| `DreamSignal` | `dream_signals` | Signal extracted from messages, memories, tools, or drafts. |
| `DreamDiaryEntry` | `dream_diary_entries` | Inner-diary style entry generated from dream signals. |
| `DreamConsolidationReport` | `dream_consolidation_reports` | Report for dream-cycle consolidation and generated proposals. |
| `DreamLock` | `dream_locks` | Runtime lock that prevents overlapping dream jobs. |
| `ExternalPageSnapshot` | `external_page_snapshots` | Snapshot of an external page read by Jina, Playwright, or CDP. |
| `ExternalInboxItem` | `external_inbox_items` | Inbound item synced from an external social or content platform. |
