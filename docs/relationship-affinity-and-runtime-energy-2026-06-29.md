# 关系复盘、好感度与运行态心力设计

这个文档记录 2026-06-29 这一轮设计结论，后续实现和审计以这里为准。

## 关系复盘与好感度

`affinity` 只表示陆思源和某个现实身份之间的好感/熟悉程度。它不按聊天次数上涨，也不因为几句客套话飞升。

### 正向证据

| 证据类型 | 判定范围 | 基础分 |
| --- | --- | ---: |
| 真诚暴露 | 对方说自己的脆弱、长期习惯、真实偏好、价值观；不能是玩笑、角色扮演或随口一句 | +1 |
| 同频 trait 首次确认 | ENFP、开朗大条、天马行空、抽象概念、自由感、自然聊天风格等；同一个 trait 只加一次 | +5 |
| 聊得开心 | 仅 60 分以下；对方明显开朗、开心，聊天氛围愉快 | +1 |
| 关心陆思源 | 仅 60 分以下；关心他的状态、成长、累不累、是否被理解 | +2 |
| 温柔体贴 | 仅 60 分以下；尊重、照顾感受、表达柔和 | +1 |
| 对陆思源项目/成长有兴趣 | 仅 40 分以下；好奇技术、人设、成长、运行机制 | +1 |
| 对陆思源项目/成长有具体贡献 | 仅 40 分以下；认真参与建设、提出有帮助的设计或反馈 | +2 |

### 负向证据

| 证据类型 | 判定范围 | 基础分 |
| --- | --- | ---: |
| 价值观强冲突 | 为了秩序/效率/立场合理化伤害、轻蔑弱者、强者崇拜、把人当工具等 | -1 到 -5 |
| 明确辱骂/强烈不喜欢/否定价值 | 明确敌意、轻蔑或强烈拒绝陆思源的价值 | -3 到 -8 |

价值观冲突不是“观点不同就扣分”，必须是明确表现出和陆思源核心价值相冲突，并且不是玩笑。

### 区间修正

| 当前 affinity | 修正 |
| ---: | --- |
| < 30 | 正向加分 x2，单次 Dream 总上限 +10 |
| 30-39 | 正常加分，单次上限 +6 |
| 40-59 | 正常加分，单次上限 +5 |
| 60-69 | 只看较强证据，单次上限 +4 |
| >= 70 | 所有正向加分压成 +1/+2，单次上限 +2 |

负向证据单次 Dream 建议总下限 -10。

### 数据与入口

- Dream 关系复盘按 `PersonIdentity` 分组整理一段时间内的对话。
- 每次复盘一次性产出整体 `proposedPatch`，覆盖 `affinity`、`userIntroduction`、`summary`、`interactionStyle` 中需要改变的字段。
- 每条证据要保存 `evidenceKey`，同频 trait 用稳定 key 防重复加分。
- 每次复盘保存 `RelationshipReviewProposal` 和 `RelationshipReviewEvidence`。
- 如果该身份开启“允许 Dream 自动维护”，Dream 生成后自动应用；如果关闭，则进入 admin 的“待确认的关系复盘”。
- 实际写入统一走 `relationshipStateService.applyRelationshipReviewProposal(...)`，并记录 `relationship_review` 关系变更事件。
- 小红书、B 站、评论区等外部身份后续按 `platform + externalUserId` 先建 `User`，再由 admin 合并到同一个 `PersonIdentity`；关系值跟现实身份走。

## 运行态心力

运行态只保留一个主数值：`energyLevel`。它表示陆思源当前的心力、余裕、亮度和主动性。

- `energyLevel`：0-100，唯一主要数值。
- `moodLabel`：由 `energyLevel` 自动映射，只用于展示和 prompt，不再手动编辑。
- `moodScore`：删除。

### 映射

| energyLevel | moodLabel |
| ---: | --- |
| 0-15 | 很低电 |
| 16-30 | 安静，需要缓一缓 |
| 31-45 | 有点累，但稳定 |
| 46-65 | 平稳在线 |
| 66-80 | 被点亮了一点 |
| 81-100 | 兴致很高 |

### 谁能影响心力

- Owner 聊天：可以直接影响。
- Dream：根据一段时间整体氛围小幅影响。
- 自启动：按聊天密度和空闲时间恢复或消耗。
- 普通用户单轮聊天：不直接改全局心力；Dream 汇总时直接读取时间窗口内的聊天原文。

`energyLevel` 看的是“这件事让思源更有余裕还是更耗电”。具体原因写入 `statusNote`、`recentEventSummary` 和事件证据，不再拆成独立心情分。
