## 1. v0.8.1 目标

让陆思源开始"看见外部世界"，但保持安全可控边界。

只做：

```text
1. 网页搜索
2. 读取已知 URL 的内容
3. 读取需要登录的平台页面（小红书评论、B站 inbox 等）
4. 截图
5. 生成摘要 / 草稿
6. 人工审核
```

不做：

```text
1. 自动发送私信
2. 自动发布内容
3. 自动点击、提交、购买
4. 自动写入正式长期记忆
5. 自动读取未授权私密信息
```

## 2. 工具选型

| 工具 | 用途 | 备注 |
|------|------|------|
| Tavily | 网页搜索（query → 结果列表） | 需要 API key，免费 1000 次/月 |
| Jina AI Reader | URL → 干净 markdown | 免费，可选 API key 提升限速 |
| Playwright | JS 渲染公开页面 + 截图 | 无头浏览器，不需要登录 |
| chrome-devtools-mcp | 连接用户已登录的 Chrome | 用于小红书等需要登录的平台 |

### 使用场景对应

| 场景 | 工具 |
|------|------|
| "搜索最近关于 XX 的新闻" | Tavily |
| "读取这篇文章" | Jina AI Reader |
| "截图这个页面" | Playwright |
| "读取我小红书的评论" | chrome-devtools-mcp |
| JS 渲染的公开页面 | Playwright |
| 静态公开页面 | Jina AI Reader |

### 工具层级

```text
fetch（原始 HTML）
  ↓ 加上解析
Jina AI Reader（干净 markdown，静态 + 部分 JS 页，免费）
  ↓ 需要完整浏览器
Playwright（完整渲染，截图，公开页面）
  ↓ 需要登录态
chrome-devtools-mcp（连接用户已登录的 Chrome）
```

## 5. 安全边界

```text
1. 所有操作默认只读，不执行写操作
2. chrome-devtools-mcp 只读取页面内容，不点击、不提交
3. 读取结果不直接写入 Memory，必须经过 Draft → 人工审核
4. CDP 连接只在用户主动触发时建立，不常驻
5. Playwright 只访问公开页面，不处理登录
```

## 6. 数据库设计

新增两张表：

### ExternalPageSnapshot

记录页面读取历史。

```sql
CREATE TABLE "ExternalPageSnapshot" (
  "id"          TEXT PRIMARY KEY,
  "url"         TEXT NOT NULL,
  "tool"        TEXT NOT NULL,  -- jina | playwright | cdp
  "content"     TEXT NOT NULL,
  "screenshotPath" TEXT,
  "createdAt"   TIMESTAMP NOT NULL DEFAULT NOW()
);
```

### ExternalInboxItem

从平台读取的 inbox / 评论条目。

```sql
CREATE TABLE "ExternalInboxItem" (
  "id"          TEXT PRIMARY KEY,
  "platform"    TEXT NOT NULL,  -- xiaohongshu | bilibili | weibo
  "sourceId"    TEXT NOT NULL,  -- 平台原始 ID，用于去重
  "type"        TEXT NOT NULL,  -- comment | mention | dm_summary
  "content"     TEXT NOT NULL,
  "authorName"  TEXT,
  "postTitle"   TEXT,
  "postUrl"     TEXT,
  "summary"     TEXT,           -- AI 生成摘要
  "draftId"     TEXT,           -- 关联草稿
  "syncedAt"    TIMESTAMP NOT NULL DEFAULT NOW(),
  UNIQUE("platform", "sourceId")
);
```

## 7. API 路由

所有路由 owner-only。

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /v1/search | 网页搜索 |
| POST | /v1/read-page | 读取 URL 内容 |
| POST | /v1/screenshot | 截图 |
| GET  | /v1/external-inbox | 查看 inbox 条目列表 |
| POST | /v1/external-inbox/sync | 手动触发平台 inbox 同步 |
| GET  | /v1/external-inbox/:id | 查看单条 inbox 条目 |

## 8. 与现有系统的关系

```text
web-search / page-reader
  ↓ 读取结果
ExternalPageSnapshot（存档）
  ↓ 可选
陆思源生成摘要 / 草稿（Draft 表）
  ↓
人工审核

external-inbox
  ↓ 读取平台评论/inbox
ExternalInboxItem（存档 + 去重）
  ↓
陆思源生成回复草稿（Draft 表）
  ↓
人工审核 → 手动发送
```
