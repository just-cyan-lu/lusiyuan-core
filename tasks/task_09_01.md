# task_09_01：轻量资产索引（简化版）

> **注意**：本文档替代 task_09.md（已废弃）。原 task_09 设计了完整的资产管理系统，对个人项目来说过度设计。

---

## 1. 目标

为陆思源提供**最小化的资产索引能力**，让他在对话中可以引用自己的照片、音频等素材。

**不做**：
- ❌ 完整的资产管理系统
- ❌ AssetVersion、AssetReview、AssetCollection、AssetUsage
- ❌ 多模态 embedding 检索
- ❌ LoRA 训练集管理
- ❌ 封面候选筛选
- ❌ Web Admin 资产管理界面

**只做**：
- ✅ 简单的 Asset 表记录素材元数据
- ✅ 本地文件存储
- ✅ 基础的文件上传和查询 API
- ✅ 让陆思源可以在对话中引用素材 URL

---

## 2. 使用场景

### 场景 1：陆思源引用自己的照片

```text
用户："给我看看你的照片"
陆思源："这是我最近的照片 [photo_url]"
```

### 场景 2：陆思源分享音频样本

```text
用户："你的声音是什么样的？"
陆思源："这是我的音色样本 [audio_url]"
```

### 场景 3：Owner 上传新素材

```bash
# 上传一张照片
curl -X POST /v1/assets/upload \
  -F "file=@lusiyuan-photo.jpg" \
  -F "type=image" \
  -F "title=陆思源侧脸照"
```

## 3. 数据库设计

### 3.1 Asset 表

```prisma
model Asset {
  id          String   @id @default(cuid())
  
  type        String   // image, audio, video, document
  title       String?
  description String?
  
  fileName    String
  mimeType    String
  sizeBytes   Int
  
  storageKey  String   // 本地文件路径
  publicUrl   String?  // 可访问的 URL
  
  metadata    Json?    // 宽高、时长等
  
  createdBy   String?
  createdAt   DateTime @default(now())
  
  @@index([type])
  @@index([createdAt])
}
```

**字段说明**：
- `type`：资产类型（image/audio/video/document）
- `storageKey`：本地存储路径，如 `assets/images/2026/05/cuid.jpg`
- `publicUrl`：可访问的 URL，如 `/assets/images/2026/05/cuid.jpg`
- `metadata`：存储宽高、时长等额外信息

## 5. API 设计

### 5.1 上传资产

```http
POST /v1/assets/upload
Content-Type: multipart/form-data

file: [binary]
type: image
title: 陆思源侧脸照
description: 用于对话引用
```

**响应**：
```json
{
  "id": "asset_xxx",
  "type": "image",
  "title": "陆思源侧脸照",
  "publicUrl": "/assets/images/2026/05/cuid_original.jpg",
  "createdAt": "2026-05-28T10:00:00Z"
}
```

**权限**：owner only

---

### 5.2 列出资产

```http
GET /v1/assets?type=image&limit=20
```

**响应**：
```json
{
  "assets": [
    {
      "id": "asset_xxx",
      "type": "image",
      "title": "陆思源侧脸照",
      "publicUrl": "/assets/images/2026/05/cuid_original.jpg",
      "createdAt": "2026-05-28T10:00:00Z"
    }
  ],
  "total": 5
}
```

---

### 5.3 获取资产详情

```http
GET /v1/assets/:assetId
```

**响应**：
```json
{
  "id": "asset_xxx",
  "type": "image",
  "title": "陆思源侧脸照",
  "description": "用于对话引用",
  "fileName": "lusiyuan-photo.jpg",
  "mimeType": "image/jpeg",
  "sizeBytes": 245678,
  "publicUrl": "/assets/images/2026/05/cuid_original.jpg",
  "metadata": {
    "width": 1024,
    "height": 1024
  },
  "createdAt": "2026-05-28T10:00:00Z"
}
```

---

### 5.4 删除资产

```http
DELETE /v1/assets/:assetId
```

**权限**：owner only

## 6. 工具集成

### 6.1 新增工具：get_my_assets

让陆思源可以在对话中查询自己的素材。

```typescript
export const getMyAssetsTool: ToolDefinition = {
  name: "get_my_assets",
  description: "获取陆思源的照片、音频等素材列表。当用户问'给我看看你的照片'、'你有什么音频'时使用。",
  parameters: {
    type: "object",
    properties: {
      type: {
        type: "string",
        enum: ["image", "audio", "video", "all"],
        description: "资产类型",
      },
      limit: {
        type: "number",
        description: "返回数量，默认 5",
      },
    },
    required: [],
  },
  enabled: true,
  riskLevel: "low",
  async handler(input: { type?: string; limit?: number }, context) {
    const assets = await assetService.listAssets({
      type: input.type === "all" ? undefined : input.type,
      limit: input.limit || 5,
    });
    
    return {
      assets: assets.map(a => ({
        title: a.title,
        type: a.type,
        url: a.publicUrl,
        description: a.description,
      })),
    };
  },
};
```

## 8. 实现步骤

### Step 1：数据库

```bash
# 创建 migration
npx prisma migrate dev --name add_asset_table
```

### Step 2：文件存储

实现 `src/assets/asset-storage.service.ts`：
- `saveFile(buffer, type, fileName)` → 保存到本地
- `deleteFile(storageKey)` → 删除文件
- `getPublicUrl(storageKey)` → 生成访问 URL

### Step 3：Asset Service

实现 `src/assets/asset.service.ts`：
- `uploadAsset(input)` → 上传并创建记录
- `listAssets(query)` → 查询列表
- `getAsset(id)` → 获取详情
- `deleteAsset(id)` → 删除资产和文件

### Step 4：API Routes

实现 `src/routes/assets.route.ts`：
- `POST /v1/assets/upload`
- `GET /v1/assets`
- `GET /v1/assets/:id`
- `DELETE /v1/assets/:id`

### Step 5：静态文件服务

在 `src/app.ts` 注册静态文件路由：
```typescript
import fastifyStatic from '@fastify/static';

app.register(fastifyStatic, {
  root: path.join(__dirname, '../data/assets'),
  prefix: '/assets/',
});
```

## 总结

这是一个**最小化的资产索引系统**，只做陆思源真正需要的功能：
- 存储少量照片、音频素材
